🛠 `Promise.all()` удобно использовать, когда нужно дождаться нескольких независимых асинхронных операций и продолжить работу только после того, как все они завершатся успешно.

Например, перед тем как показать форму оформления заказа, нужно запросить данные корзины, доступные способы доставки и способы оплаты.

Если все запросы завершатся успешно, выполнится обработчик `then()`, и мы покажем форму оформления заказа. Если хотя бы один из запросов завершится с ошибкой, промис, возвращённый `Promise.all()`, будет отклонён — сработает обработчик `catch()`: в этом случае пользователь увидит сообщение об ошибке, а форма не откроется.

```js
Promise.all([
  getCart(),
  getDeliveryOptions(),
  getPaymentMethods()
])
  .then(() => {
    showCheckoutForm()
  })
  .catch((error) => {
    showError(error)
  })
```
