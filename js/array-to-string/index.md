## Кратко

Метод `toString()` возвращает строковое представление массива.

## Пример

Создадим строку из массива:
```js

const menuItems = ['Файл', 'Редактирование', 'Просмотр']

const menuStr = menuItems.toString()

console.log(menuStr)

// Файл,Редактирование,Просмотр
```
