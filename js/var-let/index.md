## –ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ "let" –∏ "const"

–ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ `let` –∏ `const` –ø–æ—è–≤–∏–ª–∏—Å—å –≤ –≤–µ—Ä—Å–∏–∏ EcmaScript 2015 –≥–æ–¥–∞ (ES6), –Ω–∞ —Å–º–µ–Ω—É —É—Å—Ç–∞—Ä–µ–≤—à–µ–º—É `var` (–æ –Ω–µ–º –º—ã –ø–æ–≥–æ–≤–æ—Ä–∏–º –¥–∞–ª–µ–µ –≤ —Å—Ç–∞—Ç—å–µ). –û–Ω–∏ –∏–º–µ—é—Ç –±–ª–æ—á–Ω—É—é –æ–±–ª–∞—Å—Ç—å –≤–∏–¥–∏–º–æ—Å—Ç–∏ –∏ –Ω–µ —Å—Ç–∞–Ω–æ–≤—è—Ç—Å—è —á–∞—Å—Ç—å—é –≥–ª–æ–±–∞–ª—å–Ω–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞ (`window` –≤ –±—Ä–∞—É–∑–µ—Ä–µ, `global` –≤ `node`).

```js
if (true) {
  let a = 5
}

console.log(a)
// ReferenceError: a is not defined
```

```js
if (true) {
  const a = 10
}

console.log(a)
// ReferenceError: a is not defined
```

### –°–º–µ–Ω–∞ –∑–Ω–∞—á–µ–Ω–∏—è –≤ "let" –∏ "const"

–ó–Ω–∞—á–µ–Ω–∏–µ —Ö—Ä–∞–Ω—è—â–µ–µ—Å—è –≤ `let` –º–æ–∂–Ω–æ –∏–∑–º–µ–Ω—è—Ç—å.

–û–±—Ä–∞—â–µ–Ω–∏–µ –∫ –æ–¥–Ω–æ–∏–º—ë–Ω–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º `let` –ø—Ä–∏–≤–µ–¥—ë—Ç –∫ –æ—à–∏–±–∫–µ.

// SyntaxError: Identifier 'a' has already been declared

–ó–Ω–∞—á–µ–Ω–∏–µ —Ö—Ä–∞–Ω—è—â–µ–µ—Å—è –≤ `const` –∏–∑–º–µ–Ω–∏—Ç—å –Ω–µ–ª—å–∑—è, –±—É–¥—å —Ç–æ –ø—Ä–∏–º–∏—Ç–∏–≤–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∏–ª–∏ —Å—Å—ã–ª–∫–∞ –Ω–∞ –æ–±—ä–µ–∫—Ç.

```js

// TypeError: Assignment to constant variable.
```

```js

  a: 5,
}

  a: 10,
}
// TypeError: Assignment to constant variable.
```

–û–¥–Ω–∞–∫–æ –æ–±—ä–µ–∫—Ç, —Ö—Ä–∞–Ω—è—â–∏–π—Å—è –≤ `const` –º–æ–∂–Ω–æ –º—É—Ç–∏—Ä–æ–≤–∞—Ç—å. –¢–∞–∫ –∫–∞–∫ —Å—Å—ã–ª–∫–∞ –Ω–∞ –Ω–µ–≥–æ –æ—Å—Ç–∞–Ω–µ—Ç—Å—è –Ω–µ–∏–∑–º–µ–Ω–Ω–æ–π.

```js

  a: 5,
}

// { a: 10 }
```

### –ó–Ω–∞—á–µ–Ω–∏–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –¥–ª—è "let" –∏ "const"

–ó–Ω–∞—á–µ–Ω–∏–µ–º –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –¥–ª—è —ç—Ç–∏—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö —è–≤–ª—è–µ—Ç—Å—è `undefined`. –í —á–µ–º –º—ã —É–±–µ–¥–∏–ª–∏—Å—å –≤ –Ω–∞—á–∞–ª–µ —Å—Ç–∞—Ç—å–∏. –û–¥–Ω–∞–∫–æ –µ—Å—Ç—å –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –Ω—é–∞–Ω—Å—ã:
