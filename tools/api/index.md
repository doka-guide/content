–ß–∞—â–µ –≤—Å–µ–≥–æ –ø—Ä–∏ —Ä–∞–±–æ—Ç–µ —Å API –≤–µ–±-—Å–µ—Ä–≤–∏—Å–æ–≤ –≤–∞–º –±—É–¥–µ—Ç –ø–æ–ø–∞–¥–∞—Ç—å—Å—è –∏–º–µ–Ω–Ω–æ REST –∏–ª–∏ —á—Ç–æ-—Ç–æ –ø–æ—Ö–æ–∂–µ–µ –Ω–∞ –Ω–µ–≥–æ.

–í–æ–æ–±—â–µ, –Ω–µ –æ—á–µ–Ω—å –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å SOAP –∏ REST, –ø–æ—Ç–æ–º—É —á—Ç–æ REST ‚Äî —ç—Ç–æ –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π —Å—Ç–∏–ª—å, –∞ SOAP ‚Äî —Ñ–æ—Ä–º–∞—Ç –æ–±–º–µ–Ω–∞ –¥–∞–Ω–Ω—ã–º–∏. –ü–æ—ç—Ç–æ–º—É –º—ã –Ω–µ –±—É–¥–µ–º –∏—Ö —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å, –∞ –ø—Ä–æ—Å—Ç–æ —Ä–∞—Å—Å–∫–∞–∂–µ–º, –∫–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç SOAP.

–í –≤–µ–±–µ –±–æ–ª–µ–µ —á–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª–∏—Å—å XML-RPC –∏ [JSON-RPC](https://www.jsonrpc.org/specification). –ú—ã –±—É–¥–µ–º —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å –ø—Ä–∏–º–µ—Ä—ã –Ω–∞ JSON-RPC, –ø—Ä–æ—Å—Ç–æ –ø–æ—Ç–æ–º—É —á—Ç–æ JSON —Å–µ–π—á–∞—Å –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —á–∞—â–µ, –∏ –µ–≥–æ –ø—Ä–æ—â–µ —á–∏—Ç–∞—Ç—å.

–°–æ–æ–±—â–µ–Ω–∏–µ-–∑–∞–ø—Ä–æ—Å –ø–æ –ø—Ä–æ—Ç–æ–∫–æ–ª—É JSON-RPC –¥–æ–ª–∂–Ω–æ –∏–º–µ—Ç—å 3 –æ–±—è–∑–∞—Ç–µ–ª—å–Ω—ã—Ö –ø–æ–ª—è:
