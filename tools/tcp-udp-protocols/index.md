## –ö—Ä–∞—Ç–∫–æ

### –ü—Ä–æ—Ç–æ–∫–æ–ª TCP

–õ—é–±–æ–µ TCP-—Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ [–Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏—è —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è](https://hpbn.co/building-blocks-of-tcp/#slow-start). –≠—Ç–æ—Ç –ø—Ä–æ—Ü–µ—Å—Å –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Ä—É–∫–æ–ø–æ–∂–∞—Ç–∏–µ–º (handshake). –ü–µ—Ä–µ–¥ —Ç–µ–º, –∫–∞–∫ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ –º–µ–∂–¥—É –∫–ª–∏–µ–Ω—Ç–æ–º –∏ —Å–µ—Ä–≤–µ—Ä–æ–º –±—É–¥–µ—Ç —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–æ, –æ–Ω–∏ –¥–æ–ª–∂–Ω—ã —Å–æ–≥–ª–∞—Å–æ–≤–∞—Ç—å —Å—Ç–∞—Ä—Ç–æ–≤—É—é –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —á–∏—Å–µ–ª, —Å –∫–æ—Ç–æ—Ä–æ–π –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è –ø–µ—Ä–µ–¥–∞—á–∞ –¥–∞–Ω–Ω—ã—Ö. –ü–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —á–∏—Å–µ–ª —É–Ω–∏–∫–∞–ª—å–Ω–∞ –¥–ª—è –∫–∞–∂–¥–æ–≥–æ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è. –í TCP —Ä—É–∫–æ–ø–æ–∂–∞—Ç–∏–µ –ø—Ä–æ—Ö–æ–¥–∏—Ç –≤ —Ç—Ä–∏ —ç—Ç–∞–ø–∞:

1. –û—Ç–ø—Ä–∞–≤–∏—Ç–µ–ª—å –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç —Å–ª—É—á–∞–π–Ω–æ–µ —á–∏—Å–ª–æ _x_ –∏ –ø–µ—Ä–µ–¥–∞—ë—Ç –µ—ë –ø–æ–ª—É—á–∞—Ç–µ–ª—é –≤ –ø–∞–∫–µ—Ç–µ SYN. –í —ç—Ç–æ–º –∂–µ –ø–∞–∫–µ—Ç–µ –º–æ–≥—É—Ç —É–∫–∞–∑—ã–≤–∞—Ç—å—Å—è –∫–∞–∫–∏–µ-—Ç–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ —Ñ–ª–∞–≥–∏ –¥–ª—è –Ω–∞—Å—Ç—Ä–æ–π–∫–∏ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è.
2. –ü–æ–ª—É—á–∞—Ç–µ–ª—å –ø—Ä–∏–±–∞–≤–ª—è–µ—Ç –∫ —á–∏—Å–ª—É _x_ –µ–¥–∏–Ω–∏—Ü—É –∏ –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç —Å–≤–æ—ë —á–∏—Å–ª–æ _y_. –ü–æ—Å–ª–µ —ç—Ç–æ–≥–æ –ø–æ–ª—É—á–∞—Ç–µ–ª—å –ø–æ—Å—ã–ª–∞–µ—Ç –æ—Ç–ø—Ä–∞–≤–∏—Ç–µ–ª—é —á–∏—Å–ª–∞ _x + 1_ –∏ _y_ –≤ –ø–∞–∫–µ—Ç–µ SYN ACK.
3. –ó–∞–≤–µ—Ä—à–∞—é—â–∏–º —ç—Ç–∞–ø–æ–º —è–≤–ª—è–µ—Ç—Å—è –ø–µ—Ä–µ–¥–∞—á–∞ –ø–∞–∫–µ—Ç–∞ ACK –æ—Ç–ø—Ä–∞–≤–∏—Ç–µ–ª–µ–º —Å —á–∏—Å–ª–∞–º–∏ _x + 1_ –∏ _y + 1_.

–†—É–∫–æ–ø–æ–∂–∞—Ç–∏–µ –º–æ–∂–µ—Ç —Å—É—â–µ—Å—Ç–≤–µ–Ω–Ω–æ –∑–∞–º–µ–¥–ª–∏—Ç—å –ø–µ—Ä–µ–¥–∞—á—É –¥–∞–Ω–Ω—ã—Ö –ø–æ –ø—Ä–æ—Ç–æ–∫–æ–ª—É TCP –µ—Å–ª–∏, –Ω–∞–ø—Ä–∏–º–µ—Ä, –¥–∞–Ω–Ω—ã–µ –ø–µ—Ä–µ–¥–∞—é—Ç—Å—è –Ω–µ–±–æ–ª—å—à–∏–º–∏ –ø–æ—Ä—Ü–∏—è–º–∏. –ß–µ–º –±–æ–ª—å—à–µ –Ω—É–∂–Ω–æ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–π, —Ç–µ–º –º–µ–¥–ª–µ–Ω–Ω–µ–µ –æ–±—â–∞—è –ø–µ—Ä–µ–¥–∞—á–∞. –ó–∞–≥—Ä—É–∑–∫–∞ –≤–µ–±-—Å—Ç—Ä–∞–Ω–∏—Ü—ã –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–≤—è–∑–∞–Ω–∞ —Å –æ–±—Ä–∞—â–µ–Ω–∏–µ–º –∫ –±–æ–ª—å—à–æ–º—É –∫–æ–ª–∏—á–µ—Å—Ç–≤—É —Ö–æ—Å—Ç–æ–≤, —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏–µ–º —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è —Å –Ω–∏–º–∏ –∏ –∑–∞–≥—Ä—É–∑–∫–∏ —Å –Ω–∏—Ö –¥–∞–Ω–Ω—ã—Ö. –ö–∞–∂–¥–æ–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ –±—É–¥–µ—Ç —Ç—Ä–µ–±–æ–≤–∞—Ç—å —Ä—É–∫–æ–ø–æ–∂–∞—Ç–∏—è. –≠—Ç–æ –±—É–¥–µ—Ç –±–µ–∑—É—Å–ª–æ–≤–Ω–æ –≤–ª–∏—è—Ç—å –Ω–∞ —Å–∫–æ—Ä–æ—Å—Ç—å –∑–∞–≥—Ä—É–∑–∫–∏ –∫–∞–∂–¥–æ–π —Å—Ç—Ä–∞–Ω–∏—Ü—ã —Å–∞–π—Ç–∞ –≤ –±—Ä–∞—É–∑–µ—Ä–µ —É –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è.

### –ü—Ä–æ—Ç–æ–∫–æ–ª UDP
