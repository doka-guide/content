## Кратко

Элемент `<image>` позволяет вставлять изображения внутрь `SVG`-документа.

С его помощью можно отображать `PNG`, `JPEG`, `GIF`, `WebP` и даже другие `SVG`-файлы как часть `SVG`-графики.

## Пример

```html
<svg width="200" height="200">
  <image
    href="logo.png"
    width="200"
    height="200"
  />
</svg>
```

## Как пишется

Базовый синтаксис:

```html
<image
  href="image.png"
  x="0"
  y="0"
  width="200"
  height="200"
/>
```

Элемент обычно размещается внутри контейнера `<svg>`:

```html
<svg width="300" height="200">
  <image
    href="photo.jpg"
    width="300"
    height="200"
  />
</svg>
```

## Как понять

`SVG` чаще всего ассоциируется с векторной графикой: линиями, фигурами, кривыми и текстом. Однако иногда в `SVG` необходимо использовать готовые изображения, например:

- фотографии;
- логотипы;
- текстуры;
- иконки;
- другие `SVG`-файлы.

Для таких случаев используется элемент `<image>`.

Он работает похожим образом на `HTML`-тег `<img>`, но становится частью `SVG`-документа и подчиняется его системе координат.

```html
<svg width="400" height="200">
  <image
    href="cat.jpg"
    x="50"
    y="20"
    width="300"
    height="150"
  />
</svg>
```

В этом примере:

- `x="50"` смещает изображение вправо;
- `y="20"` смещает его вниз;
- `width` задаёт ширину;
- `height` задаёт высоту.

Изображение отображается как графический объект `SVG`. Это значит, что к нему можно применять:

- анимации;
- маски;
- фильтры;
- трансформации;
- обрезку через `clipPath`.

Важно помнить, что `<image>` не превращает растровое изображение в векторное. Если вставить `PNG`-файл внутрь `SVG`, он останется обычной растровой картинкой.

## Атрибуты

### `href`

Путь к изображению.

```html
<image href="logo.png" />
```

Можно использовать:
- относительные пути;
- абсолютные URL;
- Data URL;
- SVG-файлы.

### `x`

Координата изображения по оси `X`.

```html
<image
  href="image.png"
  x="50"
/>
```

### `y`

Координата изображения по оси `Y`.

```html
<image
  href="image.png"
  y="30"
/>
```

### `width`

Ширина изображения.

```html
<image
  href="image.png"
  width="200"
/>
```

### `height`

Высота изображения.

```html
<image
  href="image.png"
  height="100"
/>
```

### `preserveAspectRatio`

Управляет сохранением пропорций изображения при изменении размеров.

```html
<image
  href="image.png"
  width="200"
  height="100"
  preserveAspectRatio="xMidYMid meet"
/>
```

По умолчанию браузер старается сохранить исходные пропорции изображения.

## Что можно вкладывать внутрь

Элемент `<image>` может содержать:

- элементы анимации;
- элементы описания;
- `<animate>`;
- `<animateMotion>`;
- `<animateTransform>`;
- `<set>`;
- `<title>`;
- `<desc>`.

Например:

```html
<image
  href="logo.png"
  width="100"
  height="100"
>
  <animate
    attributeName="x"
    from="0"
    to="100"
    dur="2s"
    repeatCount="indefinite"
  />
</image>
```

## Подсказки

💡 Для современных браузеров используйте атрибут `href`. Атрибут `xlink:href` считается устаревшим.
💡 Если не указать `width` или `height`, изображение может не отобразиться.
💡 `<image>` позволяет вставлять растровую графику в `SVG`, но не преобразует её в векторную.
💡 К изображению можно применять `SVG`-фильтры, маски, анимации и трансформации.
💡 В качестве источника изображения можно использовать не только `PNG` или `JPEG`, но и другой `SVG`-файл.
