Главная

@view-transition

@view-transition — это CSS-директива, которая позволяет создавать плавные переходы между страницами. Это мощный и в то же время простой инструмент.

Первая статья

@view-transition + CSS-анимации = ❤️

Добавить свою кастомную анимацию к плавным переходам между страницами сайта очень просто. Используйте для этого простые советские CSS-анимации.
