Учёные научились выращивать зубы

Группа специалистов впервые сумела вырастить настоящий зуб в пробирке при помощи биореактора. Пока учёные выращивают зубы для мышей. Уже сегодня для выращивания зуба нужно всего две недели, после этого его отправляют в челюсть другой подопытной мыши. Почти 100% зубов приживаются.
