Предисловие о многоножках...

Тело каждой многоножки состоит из длинного туловища и округлённой головы. Обычно последняя снизу практически плоская (губоногие – исключение из этого правила), чтобы членистоногому было проще передвигаться по поверхностям. Также на голове есть пара усиков, выполняющих функцию осязательного и обонятельного органа. По-научному их называют антеннулы. Питание обеспечивают 2 пары челюстей: верхние – мандибулы, и нижние – максиллы.
