В греческой политонической орфографии тильда над гласной обозначает облечённое ударение.

В графике польского шрифта строчная буква L с тильдой над ней используется вместо буквы Ł.

В текстах на венгерском языке тильда может использоваться заменителем двойного акута.

В языках C/C++, Java, Python, Verilog и др. тильда — одноместный префиксный оператор.
