Элемент <meter>

Вносите разные значения в код выше и смотрите как меняется элемент выше

<meter max= min= value= high= low= optimum=></meter>
Кейс 1 - Атрибуты не указаны <meter></meter>

Кейс 3 - value = max (по умолчанию: min=0, max=1) <meter value="1"></meter>
Кейс 4 - value > max (по умолчанию: min=0, max=1) <meter value="5"></meter>
(если value > max, тогда value = max)
Кейс 5 - value < min (по умолчанию: min=0, max=1) <meter value="-.5"></meter>
(если value < min, тогда value = min)

Кейс 8 - value < high (по умолчанию: min=0, max=1) <meter value=".5" high=".8"></meter>
Кейс 9 - value = high (по умолчанию: min=0, max=1) <meter value=".8" high=".8"></meter>
Кейс 10 - value > high (по умолчанию: min=0, max=1) <meter value=".9" high=".8"></meter>
(обратите внимание на изменение цвета если value > high)
Кейс 11 - value < low (по умолчанию: min=0, max=1) <meter value=".15" low=".25"></meter>
(обратите внимание на изменение цвета если value < low)
Кейс 12 - value = low (по умолчанию: min=0, max=1) <meter value=".25" low=".25"></meter>
Кейс 13 - value > low (по умолчанию: min=0, max=1) <meter value=".5" low=".25"></meter>
Кейс 14 - optimum < low < high (по умолчанию: min=0, max=1) <meter low=".25" optimum=".15" high=".75" value=".5"></meter>
(обратите внимание на изменение цвета если optimum < low)

Кейс 16 - low < high < optimum (по умолчанию: min=0, max=1) <meter low=".25" optimum=".85" high=".75" value=".5"></meter>
(обратите внимание на изменение цвета если optimum > high)
