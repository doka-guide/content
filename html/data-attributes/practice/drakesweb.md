Я часто использую `data-*` атрибуты, когда нужно хранить небольшие дополнительные данные прямо в HTML. Это удобно для конфигурации компонентов, фильтрации или передачи простых параметров в JavaScript — без лишних классов или нестандартных атрибутов.

Например, у карточки может быть идентификатор и категория:

```html
<article data-id="101" data-category="design">
  …
</article>
```

В JavaScript доступ к этим данным идёт через `dataset`:

```js
const article = document.querySelector('article');

console.log(article.dataset.id); // "101"
console.log(article.dataset.category); // "design"
```

Важно помнить, что значения из `data-*` всегда приходят как строки, даже если выглядят как числа или логические значения. Если нужен другой тип, его нужно преобразовать вручную.

Я использую `data-*` для коротких строк, чисел и флагов. Для более сложных структур лучше подходит JavaScript-логика или API.
