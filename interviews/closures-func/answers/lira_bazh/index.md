Замыкание — это функция вместе со ссылкой на внешнее окружение, в котором она была создана. Проще говоря, функция имеет доступ к переменным из той области видимости, где была создана, даже тогда, когда вызывается вне этой области.

Один из частых способов использования замыканий — хранить данные внутри функции, чтобы не размещать их в глобальном контексте, но при этом сохранять к ним доступ. Замкнутые переменные никак нельзя изменить, кроме как дочерними функциями.

Базовым примером замыкания является счётчик:

```js
function createCounter() {
    // переменная во внешней функции
    let count = 0;

    return function() {
      // внутренняя функция изменяет значение count и возвращает его
      count++;
      return count;
    };
}

const counter = createCounter();
console.log(counter());
// 1
console.log(counter());
// 2
console.log(counter());
// 3
```

Таким образом, напрямую изменить значение `count` нельзя, это возможно только посредством использования функции `counter()`. Это и называется замыканием.

Подробнее о замыканиях можно прочитать в [статье об области видимости](/js/closures/#zamykanie).
