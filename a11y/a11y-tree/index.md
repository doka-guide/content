–î–µ—Ä–µ–≤–æ –¥–æ—Å—Ç—É–ø–Ω–æ—Å—Ç–∏ —Å–æ–∑–¥–∞—ë—Ç—Å—è –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω–æ —Å [DOM-–¥–µ—Ä–µ–≤–æ–º](/js/dom/#iz-chego-sostoit-dom). –ë—Ä–∞—É–∑–µ—Ä—ã –∂–¥—É—Ç, –ø–æ–∫–∞ –æ—Ç—Ä–∏—Å—É—é—Ç—Å—è —Ä–∞—Å–∫–ª–∞–¥–∫–∞ –∏ —Å—Ç–∏–ª–∏, –∞ –ø–æ—Ç–æ–º –ø—Ä–∏–Ω–∏–º–∞—é—Ç —Ä–µ—à–µ–Ω–∏–µ –æ —Ç–æ–º, —á—Ç–æ –∑–∞–±—Ä–∞—Ç—å –≤ –¥–µ—Ä–µ–≤–æ –¥–æ—Å—Ç—É–ø–Ω–æ—Å—Ç–∏. –ö—Ä–æ–º–µ —Å–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–∏—Ö —Ç–µ–≥–æ–≤ –∏ —Ç–µ–∫—Å—Ç–∞, –≤ –¥–µ—Ä–µ–≤—å—è, –∫ –ø—Ä–∏–º–µ—Ä—É, –ø–æ–ø–∞–¥–∞–µ—Ç —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ –ø—Å–µ–∫–¥–æ–∫–ª–∞—Å–æ–≤ [`::before`](/css/before/) –∏ [`::after`](/css/after/) –∏ ARIA-–∞—Ç—Ä–∏–±—É—Ç—ã.

–ë—Ä–∞—É–∑–µ—Ä—ã —Ç–∞–∫–∏–µ –∏–∑–±–∏—Ä–∞—Ç–µ–ª—å–Ω—ã–µ, —Ç–∞–∫ –∫–∞–∫ –ø—ã—Ç–∞—é—Ç—Å—è —ç–∫–æ–Ω–æ–º–∏—Ç—å —Å–≤–æ–∏ —Ä–µ—Å—É—Ä—Å—ã, –∫–∞–∫ –º–æ–∂–Ω–æ –±—ã—Å—Ç—Ä–µ–µ –æ—Ç–¥–∞–≤–∞—Ç—å –¥–µ—Ä–µ–≤–æ –≤—Å–ø–æ–º–æ–≥–∞—Ç–µ–ª—å–Ω—ã–º —Ç–µ—Ö–Ω–æ–ª–æ–≥–∏—è–º –∏ –Ω–µ –∑–∞—Å–æ—Ä—è—è—Ç—å –æ–±—ä—è–≤–ª–µ–Ω–∏—è —Å–∫—Ä–∏–Ω—Ä–∏–¥–µ—Ä–æ–≤.

## –î–æ—Å—Ç—É–ø–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã

–ü–æ—Ö–æ–∂–µ –Ω–∞ DOM-–¥–µ—Ä–µ–≤–æ, —Ç–æ–ª—å–∫–æ —Å–æ—Å—Ç–æ–∏—Ç –Ω–µ –∏–∑ HTML-—ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∞ –∏–∑ _–¥–æ—Å—Ç—É–ø–Ω—ã—Ö –æ–±—ä–µ–∫—Ç–æ–≤ (accessible object)_. –ö–∞–∂–¥—ã–π –æ–±—ä–µ–∫—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç –æ–¥–∏–Ω —ç–ª–µ–º–µ–Ω—Ç –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞ –∏ —Å–æ–¥–µ—Ä–∂–∏—Ç —Å–∞–º—É—é –≤–∞–∂–Ω—É—é –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ–± —ç–ª–µ–º–µ–Ω—Ç–µ —Å —Ç–æ—á–∫–∏ –∑—Ä–µ–Ω–∏—è –≤—Å–ø–æ–º–æ–≥–∞—Ç–µ–ª—å–Ω—ã—Ö —Ç–µ—Ö–Ω–æ–ª–æ–≥–∏–π:

- —Ä–æ–ª—å;
- –∏–º—è;
- –æ–ø–∏—Å–∞–Ω–∏–µ;
- —Å–≤–æ–π—Å—Ç–≤–∞ –∏ —Å–æ—Å—Ç–æ—è–Ω–∏—è.

–í –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ —Å–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–∏—Ö —Ç–µ–≥–æ–≤ —ç—Ç–∞ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –≤—Å—Ç—Ä–æ–µ–Ω–∞ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é, –æ–¥–Ω–∞–∫–æ —Ä–æ–ª–∏, –∏–º–µ–Ω–∞ –∏ –ø–æ–≤–µ–¥–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –º–æ–∂–Ω–æ —è–≤–Ω–æ –∑–∞–¥–∞–≤–∞—Ç—å –∏ –∏–∑–º–µ–Ω—è—Ç—å —Å –ø–æ–º–æ—â—å—é _[ARIA-—Ä–∞–∑–º–µ—Ç–∫–∏](/a11y/aria-intro/) (Accessible Rich Internet Applications)_. –≠—Ç–æ –≤—Å–ø–æ–º–æ–≥–∞—Ç–µ–ª—å–Ω–∞—è —Ç–µ—Ö–Ω–∏–∫–∞ –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –±–æ–ª–µ–µ –¥–æ—Å—Ç—É–ø–Ω–æ–≥–æ –∫–æ–Ω—Ç–µ–Ω—Ç–∞ –¥–ª—è —Å–∫—Ä–∏–Ω—Ä–∏–¥–µ—Ä–æ–≤. –†–∞—Å—à–∏—Ä—è–µ—Ç –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ HTML —Å –ø–æ–º–æ—â—å—é —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã—Ö –∞—Ç—Ä–∏–±—É—Ç–æ–≤ –∏ —Ä–æ–ª–µ–π.

–û–¥–Ω–æ –∏–∑ –≥–ª–∞–≤–Ω—ã—Ö –ø—Ä–∞–≤–∏–ª –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è ARIA ‚Äî —Å—Ç–∞—Ä–∞—Ç—å—Å—è –Ω–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å ARIA. –¢–∞–∫ —á—Ç–æ –æ–Ω–∞ –ø—Ä–∏–≥–æ–¥–∏—Ç—Å—è, –∫–æ–≥–¥–∞ –Ω–µ —Ö–≤–∞—Ç–∞–µ—Ç –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–µ–π HTML. –ö –ø—Ä–∏–º–µ—Ä—É, –¥–ª—è —Å–ª–æ–∂–Ω—ã—Ö –∏–Ω—Ç–µ—Ä–∞–∫—Ç–∏–≤–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ ‚Äî –≤–∫–ª–∞–¥–æ–∫, –≤—ã–ø–∞–¥–∞—é—â–∏—Ö —Å–ø–∏—Å–∫–æ–≤, –º–æ–¥–∞–ª—å–Ω—ã—Ö –æ–∫–æ–Ω –∏–ª–∏ –æ–ø–æ–≤–µ—â–µ–Ω–∏–π –æ–± –æ—à–∏–±–∫–∞—Ö. –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ—Ç [ARIA-–∞—Ç—Ä–∏–±—É—Ç—ã](/a11y/aria-attrs/) –∏ JavaScript —Å–¥–µ–ª–∞—é—Ç –ø–æ–≤–µ–¥–µ–Ω–∏–µ '–∏–Ω—Ç–µ—Ä–∞–∫—Ç–∏–≤–Ω–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –ø–æ–Ω—è—Ç–Ω—ã–º –∏ –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã–º.

### –†–æ–ª—å

–û–ø–∏—Å—ã–≤–∞–µ—Ç —Ç–æ, —á—Ç–æ —ç—Ç–æ –∑–∞ —ç–ª–µ–º–µ–Ω—Ç –∏ –º–æ–∂–Ω–æ –ª–∏ —Å –Ω–∏–º –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–æ–≤–∞—Ç—å. –ë—ã–≤–∞—é—Ç –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –∏ —è–≤–Ω—ã–µ —Ä–æ–ª–∏.

–í—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ —Ä–æ–ª–∏ ‚Äî —ç—Ç–æ —Ä–æ–ª–∏, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å —É [—Å–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–∏—Ö —Ç–µ–≥–æ–≤](/html/#semantika). –ù–∞–ø—Ä–∏–º–µ—Ä, —É —Å—Å—ã–ª–∫–∏ [`<a>`](/html/a/) —Å –∞—Ç—Ä–∏–±—É—Ç–æ–º `href` –µ—Å—Ç—å [—Ä–æ–ª—å `link`](/a11y/role-link/):


–ö—Å—Ç–∞—Ç–∏, –¥–∞–∂–µ —É [`<span>`](/html/span/) –∏ `<div>` –µ—Å—Ç—å –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ä–æ–ª—å [`generic`](/a11y/role-generic/), –Ω–æ —ç—Ç–æ –Ω–µ –¥–µ–ª–∞–µ—Ç –∏—Ö —Å–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–∏–º–∏ ü§Ø

–Ø–≤–Ω—ã–µ —Ä–æ–ª–∏ –∑–∞–¥–∞—é—Ç —Å –ø–æ–º–æ—â—å—é ARIA. –ö –ø—Ä–∏–º–µ—Ä—É, –ª—ë–≥–∫–∏–º –¥–≤–∏–∂–µ–Ω–∏–µ–º —Ä—É–∫–∏ `<span>` –º–æ–∂–µ—Ç —Å—Ç–∞—Ç—å —Å—Å—ã–ª–∫–æ–π –±–ª–∞–≥–æ–¥–∞—Ä—è –∞—Ç—Ä–∏–±—É—Ç—É `role="link"`, CSS –∏ JavaScript.


–û–¥–Ω–∞–∫–æ –ª—É—á—à–µ —Ç–∞–∫ –Ω–µ –¥–µ–ª–∞—Ç—å –∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ HTML –ø–æ –º–∞–∫—Å–∏–º—É–º—É. –ö–∞–∫ –º–∏–Ω–∏–º—É–º –≤–∞–º –Ω–µ –ø—Ä–∏–¥—ë—Ç—Å—è –ø–∏—Å–∞—Ç—å —Å–∫—Ä–∏–ø—Ç, —á—Ç–æ–±—ã —Å–¥–µ–ª–∞—Ç—å `<span>` –Ω–∞—Å—Ç–æ—è—â–µ–π —Å—Å—ã–ª–∫–æ–π —Å–æ –≤—Å–µ–º–∏ –Ω—É–∂–Ω—ã–º–∏ —Å–≤–æ–π—Å—Ç–≤–∞–º–∏.

–ü–æ–¥—Ä–æ–±–Ω–µ–µ —Å –∫–æ–Ω—Ü–µ–ø—Ü–∏–µ–π —Ä–æ–ª–µ–π –ø–æ–∑–Ω–∞–∫–æ–º–∏—Ç–µ—Å—å –≤ —Å—Ç–∞—Ç—å–µ ¬´[ARIA-—Ä–æ–ª–∏](/a11y/aria-roles/)¬ª.

### –ò–º—è
